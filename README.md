# IncFlow
A C++ Digital Art Program with AI Assistance
IncFlow is a custom digital art application built in C++ using modern graphics and UI libraries.
It combines traditional drawing tools with intelligent AI‑powered guidance to help artists improve their workflow, technique, and composition.

This project showcases:

real‑time rendering

image processing

custom brush engines

selection tools

multi‑layer editing

AI‑driven art tips

clean C++ architecture

It is designed as both a personal art tool and a portfolio project demonstration of software engineering, graphics programming, and user‑focused design.



Basic Features include:
Drawing canvas with real‑time brush rendering

Color picker

Reference image window

Zoom & pan

Save artwork to PNG

Clean UI (ImGui or Qt)

V2 Features:
Blur tool (box blur or Gaussian)

Lasso selection tool

Move/transform selected areas

Layer system (multiple layers + blending modes)

Undo/redo system

Custom brush textures

Navigator/minimap window

Rule‑based AI tips panel

Version 3 Features (In Development):
Full AI assistant panel with chat

Canvas analysis (color balance, composition, contrast)

Smart reference mode (palette extraction, shape guidance)

Style suggestions

Lighting and anatomy tips

Brush recommendations


Ai assistant: includes an AI assistant designed to help artists improve their work.
Depending on the version, it can:

give real‑time drawing tips

analyze the canvas

suggest palettes

highlight composition issues

provide reference‑based guidance

This feature is built to demonstrate AI integration and creative UX thinking.


Stack:

C++

CMake

SFML / Raylib (graphics)

Dear ImGui / Qt (UI)

stb_image / stb_image_write (image loading & saving)

Custom rendering + brush engine

AI model (in development)
